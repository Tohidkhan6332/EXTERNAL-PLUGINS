 ##  Command Usage

 1. To install A Plugin use 
 ```SH
  .install <plugin url>
 ```
2. To Delete A Plugin use
 ```SH
  .remove <plugin name>
 ```
3. To See all available Plugins use
 ```SH
  .listplugins
 ```
